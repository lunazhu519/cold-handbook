# 冷手册 (Cold Handbook)

一个交互式感冒自查与处理指南。帮助你快速判断感冒类型、所处阶段，并提供针对性的中西医处理方案。

## 功能特性

### 📖 手册
系统化的感冒知识库，包含六个核心模块：
- **第一步判断** - 快速区分感冒类型（流感/过敏/普通感冒）
- **进程阶段** - 了解感冒的不同发展阶段及特点
- **西医实操** - 西医诊疗方法和建议
- **中医实操** - 中医调理思路和方案
- **用药参考** - 常用药物及用法说明
- **儿童专项** - 针对儿童感冒的特殊处理建议

点击展开任意章节查看详细内容。

### 🔍 自查
智能症状诊断工具，帮助你快速定位：
- 选择当前症状
- 自动判断感冒类型（流感/过敏/普通感冒）
- 识别当前处于哪个阶段
- 区分风寒还是风热
- 获得个性化处理方案

### 📝 记录
云端感冒日志系统：
- 记录每次感冒的详细信息（时间、人物、类型、症状、处理方式、用药、观察）
- 建立个人感冒历史档案
- 支持导出功能 - 一键复制所有记录到 Notion 或其他平台
- 零云端占用成本 - 数据以易迁移格式存储

### 📓 笔记
实时心得记录：
- 随时记录新发现和个人心得
- 云端自动保存
- 方便日后查阅和改进

## 使用方法

1. 打开 `index.html` 在浏览器中查看
2. 选择需要的功能模块开始使用
